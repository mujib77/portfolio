# Portfolio

This is my personal portfolio website built using React, Vite, and Tailwind CSS.

I made this to have a simple place to showcase what I’m building and learning as a developer. It’s not meant to be over-designed, just clean and focused on content.

## Tech Stack

- React
- Vite
- Tailwind CSS

## Features

- Minimal and clean UI
- Sections for about, skills, and projects
- Fully responsive layout
- Deployed online

## What I learned

While building this, I got more comfortable with:
- structuring React components properly
- working with Tailwind for styling
- setting up and deploying a frontend project
